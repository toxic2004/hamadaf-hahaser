# המדף החסר

יישום אינטרנטי בעברית ובכיוון ימין לשמאל לניהול ספרים מבוקשים. הנתונים נשמרים ב Supabase עם הפרדה לפי משתמש, ומסך הכניסה נשאר נקודת הכניסה לכל כלי המערכת.

## יכולות

- ניהול ספרים במצבי מחפש, משא ומתן, מחכה לתשובה, הושג וסל מחזור
- איתור לפי ISBN וזיהוי כריכה מצילום או קובץ
- עדיפות רגילה, גבוהה או דחופה, ספר חובה ורשימת מועדפים
- סטטיסטיקות חודשיות, שנתיות ולכל התקופה עם סימון מפורש של מידע חסר
- ייצוא XLSX אמיתי בעברית, עם גיליונות, טבלאות, מסננים והקפאת כותרות
- השוואת הצעות חדשות ויד שנייה שהוזנו ידנית ממקורות מורשים
- תיעוד יומי של המחיר הזול, היסטוריה לשנה והתראות בתוך המערכת
- תשתית אופציונלית לדוח בוקר ולמייל, ללא מפתחות או סודות במאגר

## פיתוח ובדיקות

נדרשת Node.js בגרסה 20 ומעלה.

```bash
npm ci
npm run format:check
npm run lint
npm run typecheck
npm test
npm run build
```

ה build הסטטי נכתב לתיקייה `dist`. הוראות הפעלת המיגרציות, פונקציית ההתראות והסודות החיצוניים נמצאות בקובץ `UPGRADE-DEPLOYMENT.md`.

## מגבלות מכוונות

המערכת אינה עוקפת התחברות, CAPTCHA, הגנות אתר או תנאי שימוש. במקורות שאין להם חיבור חוקי ויציב, החיפוש נפתח באתר המקור והמידע מוזן ידנית. אין חיבור ל ChatGPT. מיילים אינם נשלחים עד לפריסה ולהגדרה מפורשת של שירות המייל.
